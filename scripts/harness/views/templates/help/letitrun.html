Let's run the simulation on automatic by pressing the play button: You can stop at any time with the stop button.

ArraySource1 will loop round its three classes, presenting each to the PSOM.

PSOM3 will initially grow to accept the new class and then over time it will push dissimilar classes apart and break long links. Neurons without any links are removed. Eventually, you should have three tight neuron clusters floating in the middle.

ScalarSink2 will still be showing the last error, probably below 0.01.
